🏨 ReviewInsight

AI-powered Hotel & Restaurant Review Analytics Platform

ReviewSense AI is a Streamlit-based application that transforms Google Reviews into actionable business insights using Natural Language Processing (NLP) and sentiment analysis.

✨ Features
📂 Upload CSV or Excel review data
😊 Hybrid Sentiment Analysis
🏨 Hotel Health Score (0-100)
📊 Interactive Dashboard & Charts
☁️ Review Word Cloud
🔍 Review Search Engine
🤖 AI-based Business Insights
📥 Export Processed Data
🛠️ Tech Stack
Python
Streamlit
Pandas
Plotly
NLTK (VADER)
WordCloud
Matplotlib
🚀 Installation
git clone https://github.com/yourusername/ReviewSense-AI.git

cd ReviewSense-AI

pip install -r requirements.txt

streamlit run app.py
📁 Input Format

Supported file types:

CSV (.csv)
Excel (.xlsx)

Required columns:

Review
Review2 (optional)
rating
